# MicrofoneParaPlantas
Este codigo pertence ao projeto que fiz no meu canal de YouTube com o objetivo que dar voz ás plantas de forma a saber quando devo regar ou deixa-la ao sol.

Link do video : https://youtu.be/oBSLlSilqAA

Guia instructables : https://www.instructables.com/editInstructable/edit/EPTFF8LKHKK6IKW

Este códgio está dividido em 2 e deve ser usado com 2 arduinos diferentes, o código Smart_Pot.ino deve ser usado para ler os sensores e controlar a coluna, e o código I2C_Display.ino serve para pintar o display OLED. Não consegui colocar tudo num só arduino devido a imcapatibilidade de bibliotecas e falta de memória.


Os componentes usados neste projecto são os seguintes :

Sensor de humidade de solo : https://www.ebay.com/itm/NEW-Analog-Capacitive-So...

Sensor de luz LDR : https://www.ebay.com/itm/50pcs-Photoresistor-LDR-C...

Resistências de 10k : https://www.ebay.com/itm/100Pcs-1-4W-0-25W-Metal-F...

Leitor SPI SD Card : https://www.ebay.com/itm/Arduino-Micro-SD-card-min...

Display OLED I2C : https://www.ebay.com/itm/0-96-I2C-IIC-Serial-128X6...

Arduino Nano : https://www.ebay.com/itm/2-5-10PCS-USB-Nano-V3-0-A...
